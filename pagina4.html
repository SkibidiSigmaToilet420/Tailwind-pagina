Tablas
Las tablas son la estructura fundamental de una base de datos. Cada tabla almacena datos sobre una entidad específica, como usuarios, productos o transacciones. Las tablas se componen de filas y columnas.


Registros y campos (Filas y columnas)
Cada fila en una tabla representa un registro único. Por ejemplo, en una tabla de usuarios, cada fila podría representar un usuario diferente.
Las columnas definen los atributos de los datos almacenados en la tabla. Por ejemplo, en una tabla de usuarios, las columnas podrían ser ID, Nombre, Correo Electrónico, etc.


Primary Key
Una clave primaria es un campo (o combinación de campos) que identifica de manera única cada registro en una tabla. Por ejemplo, el ID de usuario.


Foreign Key
Una clave primaria es un campo (o combinación de campos) que identifica de manera única cada registro en una tabla. Por ejemplo, el ID de usuario.


Candidate Key
Una clave candidata es una superclave mínima. Esto significa que es una combinación de atributos que puede identificar de manera única cada fila en una tabla, y no contiene atributos redundantes
